# API Server

Simple Rest API using gin(framework) & gorm(orm)

## Endpoint list

### Profiles Resource

```
GET    /api/profiles
GET    /api/profiles/:id
POST   /api/profiles
PUT    /api/profiles/:id
DELETE /api/profiles/:id
```

### Emails Resource

```
GET    /api/emails
GET    /api/emails/:id
POST   /api/emails
PUT    /api/emails/:id
DELETE /api/emails/:id
```

### Companies Resource

### Users Resource
